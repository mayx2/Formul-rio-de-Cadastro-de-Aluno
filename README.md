
# Formulário Cadastro de Aluno 

Formulário simples para cadastro de aluno, utilizando **HTML**, **Bootstrap 5** e **JavaScript** para validações.



##  Funcionalidades

* Validação dos campos obrigatórios
* Validação personalizada:

  * **Matrícula**: exatamente 14 caracteres
  * **CEP**: somente números e exatamente 8 dígitos
* Preenchimento automático da **data de ingresso**
* Mensagem de sucesso ao enviar
* Layout responsivo com Bootstrap



##  Como usar

1. Abra o arquivo **index.html** no navegador.
2. Preencha os campos do formulário.
3. Clique em **Salvar** para validar e ver a mensagem de sucesso.



##  Tecnologias

* HTML
* CSS
* JavaScript
* Bootstrap 5
